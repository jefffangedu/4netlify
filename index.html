電子電路課程
八年級上學期
Lab01 - 認識電路
課程大綱
暖身活動
什麼是電？家裡的電燈、玩具車為什麼會動？
教學重點
介紹電池、LED、電阻、馬達等基本元件。透過實際組裝，讓學生了解電壓、電流與電阻的關係。
實驗設計
LED 與電阻
學生嘗試直接將 LED 接上電池，觀察燒毀現象。接著，加入電阻保護 LED，讓燈光成功亮起。
馬達與電阻
學生將馬達接上電池，觀察轉動。接著，加入不同阻值的電阻，觀察馬達轉速的變化。
Block 程式碼： 無，本單元不使用 Arduino。
Lab02 - 初見 Arduino
課程大綱
暖身活動
機器人、智慧家電為什麼可以自動運作？
教學重點
介紹 Arduino 是電路的大腦，可以用程式來控制。學習如何使用 Arduino 的數位輸出來控制 LED 燈的亮與滅。
實驗設計
元件連接
將 LED 和電阻連接到麵包板，再將電阻連接到 Arduino 的數位腳位（例如 Pin 13）。
程式控制
使用積木程式，設定 Pin 13 輸出 高電位（HIGH） 和 低電位（LOW），並加上等待時間，讓 LED 燈每秒閃爍一次。
Block 程式碼

設定 Pin 13 為高電位 等待 1 秒 設定 Pin 13 為低電位 等待 1 秒 重複執行
Lab03 - 紅綠燈
課程大綱
暖身活動
紅綠燈是怎麼運作的？每個燈亮多久？
教學重點
學習如何使用 麵包板 搭建多個元件的複雜電路。練習控制多個數位輸出來實現複雜的邏輯。
作業
模仿真實十字路口，設計雙向紅綠燈控制。
實驗設計
電路搭建
在麵包板上分別連接紅、黃、綠三顆 LED，並接上各自的電阻，再將它們連接到 Arduino 的不同數位腳位。
程式控制
使用積木程式，依照實際紅綠燈的順序與時間（例如：紅燈亮5秒、綠燈亮5秒、黃燈亮2秒）來控制 LED。
Block 程式碼

紅燈亮、其他燈滅 等待 5 秒 紅黃燈一起亮 等待 1 秒 綠燈亮、其他燈滅 等待 5 秒 黃燈亮、其他燈滅 等待 2 秒 重複執行
Lab04 - 按鈕控制燈光
課程大綱
暖身活動
遊戲機的按鈕、電燈開關為什麼能發出指令？
教學重點
學習 數位輸入 的概念。認識上拉電阻的作用，並使用按鈕作為輸入裝置，讓學生了解如何讀取按鈕的狀態。
實驗設計
電路搭建
將按鈕連接到 Arduino 的數位輸入腳位。同時連接上拉電阻，確保按鈕未按下時腳位處於穩定狀態。
程式控制
使用「如果...那麼...否則...」的邏輯積木，讀取按鈕的狀態，並根據狀態控制 LED 燈的亮與滅。
Block 程式碼

如果 讀取按鈕腳位為高電位，那麼 LED 亮。 否則 LED 滅。
作業
閃爍警示燈： 製作一個按鈕控制的閃爍警示燈。按下按鈕，LED 會開始以特定頻率閃爍；再次按下，閃爍停止，LED 熄滅。
Lab05 - 可變電阻控制燈光亮度
課程大綱
暖身活動
家裡的檯燈、音響如何調整亮度或音量？
教學重點
介紹 類比輸入 的概念，它能讀取連續變化的數值。學習使用可變電阻讀取 0-1023 的數值變化。
實驗設計
電路搭建
將可變電阻連接到 Arduino 的類比輸入腳位（例如 A0）。
程式控制
使用積木程式，讀取 A0 腳位的數值，並將這個數值顯示在序列監視器上。轉動可變電阻，觀察數值的變化。
Block 程式碼

讀取類比腳位 A0 顯示在序列監視器
作業
利用光敏電阻製作夜燈
電路搭建： 使用光敏電阻取代可變電阻，組成一個分壓電路。
程式控制： 讀取光敏電阻的數值。設定一個臨界值，如果數值低於此值（代表環境變暗），則點亮 LED；反之則關閉 LED。
Lab06 - PWM 控制燈光亮度
課程大綱
暖身活動
演唱會的舞台燈光，為什麼可以呈現出各種顏色和明暗變化？
教學重點
認識 RGB 三色 LED，它由紅、綠、藍三種顏色組成。學習如何使用 Arduino 的 PWM 腳位來控制輸出的電位，從而控制 LED 的亮度。
實驗設計
電路搭建
將 RGB LED 的三隻腳分別連接到 Arduino 的 PWM 腳位（例如 Pin 9、10、11）。
程式控制
使用「設定 PWM 輸出」的積木，輸出 0-255 的數值。嘗試調整紅、綠、藍三種顏色的數值，混合出不同的顏色。
Block 程式碼

設定 PWM 輸出 9 到 亮度變數 亮度變數 +1 如果 亮度變數 > 255 ...
作業
呼吸燈，並帶有顏色變化
程式控制： 使用變數來控制 PWM 數值，讓 LED 亮度從 0 逐漸增加到 255，再逐漸減少到 0。
顏色變化： 在亮度變化的同時，逐步改變紅、綠、藍三種顏色的比例，實現呼吸燈與變色的效果。
Lab07 - WS2812B 三色燈條
課程大綱
教學重點
簡介 WS2812B 燈條的原理，它是一個可編程的 LED 燈，只需要一條訊號線就可以控制多個燈珠。
示範設計
電路搭建
連接 WS2812B 燈條的電源、接地和訊號線到 Arduino。
程式示範
由於 Tinkercad 暫不支援相關積木，老師可直接用 Arduino IDE 展示程式碼，讓學生觀察燈條產生的炫麗燈光效果，作為知識的延伸。
期末作業
主題：製作一個屬於自己的燈具
要求：綜合運用 Lab01 至 Lab06 所學的知識，設計一個具有以下任一功能的燈具：
聲控燈
使用麥克風作為輸入，讓燈光隨著聲音大小變化。
感應燈
使用人體紅外線感應器，當有人經過時自動開燈。
智慧夜燈
結合光敏電阻和按鈕，實現手動開關和自動感應。
裝飾燈
利用可變電阻或按鈕，製作一個可以變換顏色和亮度的裝飾燈。
